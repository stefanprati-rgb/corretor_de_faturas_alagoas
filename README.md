# 🔍 Corretor de Faturas Alagoas

Sistema automatizado para análise e correção de faturas de energia elétrica do estado de Alagoas.

## 📋 Descrição

Este projeto fornece uma interface web para upload e processamento de faturas de energia em formato PDF, identificando possíveis inconsistências e erros de cobrança.

## 🚀 Funcionalidades

- Upload de faturas em PDF via drag & drop ou seleção de arquivo
- Interface moderna e responsiva
- Processamento automatizado de documentos
- Análise de dados de consumo e valores

## 🛠️ Tecnologias

- HTML5
- CSS3 (design moderno com gradientes)
- JavaScript vanilla
- Design responsivo mobile-first

## 📦 Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/stefanprati-rgb/corretor_de_faturas_alagoas.git
```

2. Abra o arquivo `index.html` no seu navegador

3. Faça upload da sua fatura em PDF

4. Clique em "Processar Fatura" para análise
